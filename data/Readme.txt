All the Data files used in developing the model and validations can be found here.
Data.xlsx has all the training, test and validation dataset.
ImPy.xlsx has the Imidazopyridine amides used in validations.
BPip.xlsx includes all the structures assessed in Benzyl piperazine ureas section.

The evaluation results obtained for the test and validation datasets is given in Results.xlsx and the Y-scrambling results are in YScrambling_100.xlsx
